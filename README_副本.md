# AIGC-Prompt-Library

这是一个面向 AIGC 设计师的个人 Prompt 库与设计知识库，用来沉淀商业视觉生成、产品精修、包装提案、IP 延展、3D 渲染、海报 KV、视频分镜与案例复盘方法。

本仓库的核心原则是：先锁定不变项，再明确生成目标，最后用材质、光影、构图、负面词和修改指令控制结果。

## 使用场景

- 电商产品图、食品白底图、去背景白底图生成与修正
- 酸奶、牛奶、巧克力、饮料、零食等食品商业摄影 Prompt
- 包装设计、包装提案、产品系列化视觉探索
- IP 角色三视图、表情延展、周边设计与角色一致性控制
- 2D 插画转 3D 渲染、IP 手办风格、产品级渲染展示
- 品牌海报 KV、节日主视觉、社交媒体视觉提案
- 人像高清重建、商业修图、材质增强、画面融合
- 视频分镜、广告脚本视觉化、项目汇报与作品集复盘
- 失败案例归档，记录问题、原因与下一次修改方向

## 我的 AIGC 设计工作流

1. 明确任务目标：主图、KV、包装提案、IP 延展、修图、分镜或作品集案例。
2. 确认不变项：Logo、文字、包装形状、构图、比例、角色五官、人物身份、色彩关系。
3. 写出生成目标：商业摄影、白底主图、包装方案、三视图、3D 渲染、节日海报或视频分镜。
4. 补充视觉控制：材质、光影、色温、镜头、背景、构图、输出比例、留白区域。
5. 加入负面词：禁止变形、改字、换脸、塑料感、元素过满、边缘脏、主体漂移。
6. 小步迭代：每次只改一个问题，并记录修改指令、效果评分和下一步。
7. 复盘沉淀：把成功 Prompt、失败原因、可复用修饰词归档到对应目录。

## 持续更新说明

这个仓库会随着真实项目持续更新。每次新增 Prompt 时，建议同步记录适用模型、输入素材、生成比例、修改轮次、失败原因和最终评分。长期目标不是堆积句子，而是形成一套可复用、可复盘、可展示的 AIGC 设计工作方法。
