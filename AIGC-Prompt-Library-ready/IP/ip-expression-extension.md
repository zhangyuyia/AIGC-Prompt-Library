# IP 表情与动作延展

## 使用场景

适用于 IP 表情包、动作姿势、品牌活动角色延展和社交媒体视觉。

## 原始需求

在保持 IP 角色一致的前提下，生成不同表情、动作和简单道具组合。

## Prompt

基于原有 IP 角色进行表情与动作延展，严格保持角色脸型、五官比例、头身比、身体结构、核心轮廓、主配色、服装样式和整体风格完全一致。仅改变表情、动作、手势和少量场景道具，使角色更适合表情包、品牌海报和社交媒体传播。角色轮廓清晰，表情自然有亲和力，动作明确不过度夸张，画面干净，适合系列化输出。

## 负面词

不要变脸、换角色、改变主配色、改变头身比、改服装、过度复杂道具、表情失控、角色比例崩坏。

## 修改指令

- 角色不像原图：加强脸型、五官比例、头身比和核心轮廓不变。
- 动作不清楚：减少道具，强化手势和身体姿态。
- 系列不统一：统一线条、材质、光影和背景。

## 注意事项

IP 延展只改变“动作和情绪”，不要让模型重新设计角色。

## 效果评分

8/10：适合表情包和品牌 IP 延展。
