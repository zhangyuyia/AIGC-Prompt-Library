# IP 角色标准三视图

## 使用场景

适用于根据正面 IP 形象生成正视图、纯侧视图和纯背视图。

## 原始需求

在不改变角色设计的前提下，生成可用于建模、提案和角色设定展示的三视图。

## Prompt

基于原始正面 IP 形象生成标准三视图：正视图、纯侧视图、纯背视图。严格保持角色脸型、五官比例、头身比、身体结构、核心轮廓、服装、配色、表情和所有设计细节完全一致，不改变任何核心设计。纯侧视图必须是严格 90 度正侧面，只显示一条腿和一只脚，禁止第二条腿、重影或透视穿插。纯背视图必须是完全背面，不出现侧脸或角度偏移。统一 3D 渲染风格，白底横向排布，高清产品级展示效果。

## 负面词

不要重设计、换角色、改变脸型、改变头身比、改变服装、改变颜色；侧视图不要斜侧、双腿、重影；背视图不要露侧脸。

## 修改指令

- 侧视图错误：强调 strict 90-degree orthographic side view, one leg and one foot only。
- 背视图错误：强调完全背面，不可出现侧脸或斜角。
- 角色漂移：重新列出五官、头身比、轮廓、服装和配色不变。

## 注意事项

三视图最容易出现侧面角度不准和角色漂移，必须把约束放在 Prompt 前半段。
