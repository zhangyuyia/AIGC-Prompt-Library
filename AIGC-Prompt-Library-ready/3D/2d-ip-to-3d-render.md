# 2D IP 转 3D 手办渲染

## 使用场景

适用于将 2D 插画 IP 转化为 3D 渲染、手办效果、周边展示图。

## 原始需求

不改变原有 IP 设计，只增加三维体积、材质和光影表现。

## Prompt

将原始 2D 插画 IP 转化为高质量 3D 渲染效果，在完全保持原有设计不变的前提下进行。不改变角色造型、比例、轮廓、颜色、表情、服装与任何细节，仅增加三维体积感、圆润边缘、干净材质和真实光影。加入柔和高光与接触阴影，使角色具有类似高质量 IP 手办的真实展示效果。白色背景，高清、清透、细节丰富、无风格偏移。

## 负面词

no redesign, no change in shape, no change in proportions, preserve original design exactly, only add volume and lighting, no extra accessories, no identity drift.

## 修改指令

- 形状变了：强调 preserve original design exactly。
- 3D 感不足：增加体积、圆润边缘、柔和高光和接触阴影。
- 角色过度写实：保持原插画比例和品牌 IP 风格。

## 注意事项

2D 转 3D 的关键不是重新设计，而是保形后增加体积和材质。

## 效果评分

8.5/10：适合 IP 3D 化和周边提案。
