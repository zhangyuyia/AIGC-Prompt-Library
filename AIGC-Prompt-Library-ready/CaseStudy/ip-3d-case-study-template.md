# IP 3D 化案例复盘模板

## 使用场景

适用于 IP 三视图、2D 转 3D、表情延展、周边开发等案例整理。

## 原始需求

把 IP 从平面延展到 3D 或系列化视觉的过程整理成案例。

## Prompt

请将这个 IP 视觉项目整理为作品集案例，包含：项目背景、角色设定、核心识别特征、必须保持不变的设计项、3D 转化或延展策略、Prompt 关键词、迭代问题、修正方法、最终输出和商业应用场景。语言需要突出角色一致性、品牌识别、系列化能力和可生产性，适合用于作品集页面或工作汇报。

## 负面词

不要只描述画面效果、不要忽略角色漂移问题、不要过度夸大、不要使用空泛形容词、不要缺少应用场景。

## 修改指令

- 角色一致性表达不足：补充脸型、五官、头身比、轮廓、配色和服装。
- 商业价值不足：补充表情包、海报、周边、建模参考等用途。
- 复盘太平：加入失败点和具体修正指令。

## 注意事项

IP 案例的重点是“一致性控制”，不是单张图好看。

## 效果评分

8/10：适合 IP 设计作品集复盘。
